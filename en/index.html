svgicon

SVG icon components and tool set

Quick Start →

Support multiple js frameworks

Vue 2.x, Vue 3.x, React >= 16.8 icon components are provided by default, and components that support other frameworks can be written through @yzfe/svgicon

Load with webpack loader

Load SVG files into icon data or icon components through webpack loader (@yzfe/svgicon-loader), and you can customize the generated code.

Multiple features

Supports multiple colors, including gradients; Supports set fill and stroke attributes at the same time; supports original colors, and can modify a certain color; Supports zoom, animation etc.

Icon Preview

In any folder, use @yzfe/svgicon-viewer to view the SVG icon effect
